# lovelypet
一款萌宠类的go后端

## 业务1：每日心情
1.每天只能发表一条心情记录

2.可以查询、删除以前发表的心情

3.封装gorm(mysl)

4.封装redis(缓存)

5.封装mongodb(nosql型)

6、jwt+https

7、rpc+grpc

8、一些数据类型，os，io工具类

9、oauth第三方授权

10.一些常用的加解密，base64，对称加密

11.go module管理

12.一些主流的进程框架，比如rabbitmq或者nsq

13.一些主流的工具，诸如zap(log日志),crontab(定时任务)

14.一些产品需求，诸如请求频率控制，websocket
